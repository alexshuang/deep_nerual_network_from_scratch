# Deep Nerual Network From Scratch

随着深度学习的大热，各种模型层出不穷，说是全民算法也不为过。授人以鱼不如授人以渔，《Deep Nerual Network From Scratch》通过自己动手用Python重写FC、RNN、GRU、LSTM、CNN、BatchNorm、初始化等核心模块的方式，让你面对全新的模型也做到心中不慌。
